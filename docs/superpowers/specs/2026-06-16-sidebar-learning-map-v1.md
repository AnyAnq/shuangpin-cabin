# 侧边栏学习地图 V1

## 目标

左侧栏从静态图标升级为学习地图，串联练习、键位对照、记录、课程和设置。V1 先落地真实路由、速查型键位对照页和错题复盘型记录页。

## V1 范围

- 练习：保留现有首页和输入体验。
- 键位对照：提供当前双拼方案的键位速查表，支持按键、声母、韵母搜索。
- 记录：读取当前方案错题，展示错因、目标字、按错键、次数、连续正确和最近错误时间。
- 侧边栏：练习、键位对照、记录为可访问路由；课程和设置保留为 Phase 2 入口。

## Phase 2 方向

- 课程：短训练路径，不做长文章。建议分为声母熟悉、韵母定位、高频组合、易混专项和诗句实战，每个课程能进入对应练习。
- 设置：集中放提示强度、默认方案、是否显示汉字下方键帽、是否显示虚拟键盘和清空本地数据。

## 设计原则

- 侧边栏保持深暖色浮动图标栏，不展开成后台菜单。
- 新页面沿用暖色面板、绿色键帽和克制卡片样式。
- 记录页优先反哺练习，不做完整趋势图。
