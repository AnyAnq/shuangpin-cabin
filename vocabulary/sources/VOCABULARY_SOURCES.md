# 词库筛选规则与来源说明

## 当前版本

这一版已经结合 GitHub 开源词库扩容，不再只是少量手工 starter 词。

## 使用的开源来源

1. THUOCL，MIT License。使用 data 目录下 IT、财经、成语、地名、饮食、法律、医学、诗词、汽车等分类词库。
2. Jieba，MIT License。使用 extra_dict/dict.txt.big 作为日常高频词基础来源。

## 筛选规则

1. 只保留纯中文词条。
2. 只保留 2 到 4 个汉字的词，避免练习组过长或过碎。
3. 删除明显机构名、公司后缀、新闻署名和不适合练习的固定噪声。
4. 包内去重，registry 的 entryCount 与实际 entries 数量一致。
5. 根据源词频或源排序保留高频优先项。
6. 将源词频归一为 1-100 的 weight，越靠前越优先出题。
7. 每个词条保留 source 字段，便于后续追踪来源。

## 后续扩展建议

1. 增加人工黑名单，过滤不想出现的人名、地名或行业词。
2. 增加分级字段，例如 beginner/intermediate/advanced。
3. 对易混音词库使用拼音分析进一步精筛，而不是只靠字形规则。
4. 加入 checksum，防止下载文件被误改。
